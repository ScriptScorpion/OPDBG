## OPDBG (Over Powered DeBuGers) - Collection of useful tools for debugging and Reverse Engineering

# Requirements:
* For Disassembler: needed _libncurses-dev_ package for Linux, or _mingw-w64-x86_64-ncurses_ for Windows.
* g++
* make

# How to install:
* Run make in directory of what tool you want to use
